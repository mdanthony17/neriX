Extending tkdumper

The tkdumper program supports dynamic extension. Dynamic extension means that the functionality of the program can be extended while it is running. This is possible because the program is completely written in Tcl/Tk.

The following can be extended:

  • Packet definitions - Additional packet types can be made known to tkdumper by using the File->Packet Defs... menu command to read in a packet definition. Follow the link above to see what the format of a packet definition file is.
  • Plugin packet formatters - The body of a packet can be formatted in whatever way you want by loading and registering packet formatting plugins. Plugins are loaded via the File->Plugin... menu command. To see how to write a packet formatting plugin, follow the link above.
