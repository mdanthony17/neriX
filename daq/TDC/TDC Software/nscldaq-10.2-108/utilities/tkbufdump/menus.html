The tkdumper menubar

The tkdumper menubar at he very top of the application provides access to the commands you can give the program. There are three menus in the menubar:

  • File
  • Filter
  • Help

    • The File menu

    The file menu offers the following commands:

    • Open... - this command opens a dialog that allows you to connect tkdumper to an event source. Until this is done, the application is not able to display events. For more information on the Open command, see the Open dialog's context sensitive help.
    • Packet Defs... - this command opens a file chooser dialog. Use this dialog to select a file that contains packet definitions. For more information about the format of this file see packet definition files. Once read, a packet definition file extends the set of packet types that are recognized and formatted as packets in physics events.
    • Plugin... - this command opens a file chooser dialog. Use this dialog to select a file that contains plugin Tcl code. Plugin code can be used to format packets in physics events in whatever manner you choose. For more information see Plugin Code.
    • Exit - Exits the program.

    The Filter menu

    The filter menu provides the following commands:

    • Filter Types... - allows you to specify a data type filter. data type filters restrict the set of event types that will be displayed when you request a sample. Clicking this menu item will bring up a dialog that prompts for the filter specification. Filter dialog context sensitive help provides information about this dialog.
    • Remove Filter - Removes any defined filter.

    The Help Menu

    The help menu provides the following commands:

    • About... displays information about the current version of tkdumper
    • Topics.. Accesses the online help system you are reading now.
