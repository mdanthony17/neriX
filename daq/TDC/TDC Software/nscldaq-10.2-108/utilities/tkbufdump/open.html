Opening a data source

This dialog allows you to open a data source for the tkdumper program. Two data items are required and should be filled in in the dialog above:

Host:
This is the computer that is the source of data for tkdumper. The default is localhost which is the system on which you are running tkdumper.
User:
This is the username of the account that is running the readout program on the host computer. This defaults to the name of the user running tkdumper.

Fill in the two entries (usually you will only need to modify the Host field). And:

  • Click Ok to accept and connect to that data source.
  • Click Cancel to dismiss this dialog and do nothing.
