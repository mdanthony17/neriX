Creating a data filter

Typically the data coming from a readout program is dominated by physics events. This can make it hard to see other types of events if this is desired. Type filters allow you to select which event types are displayed when requesting to see a new event.

The dialog allows you to specify both the standard event types and any user defined event types you might have created.

To add a standard event type to the set of accepted event types, either highlight it in the Item Types list box and click the right arrow or double click the item in the Item Type listbox. Custom event types can be added by id number. To do this type the ID number in the User ItemType entry box and click the Add button.

If you have accidently added an even type to the Selected Item types list you can remove it. Either select the item in the Selected Item types list box and click the left arrow button or double click the item type in the Selected Item types listbox.

When you are done:

  • Click OK to apply the filter described in the Selected Item types box
  • Click Cancel to dismiss the dialog without doing anything.

Filters can be removed via the Filter->RemoveFilter menu command.
