Overview

The epicsdisplay program's primary window is a table of channel values. Each line in the table contains the name of the channel, channel value, units of measure and the comment text supplied in the configuration file for that channel. THe bottom part of this window is an optional chart recorder that can show the time evolution of the values of a set of channels.

The user interface supports:

  • Dynamically adding channels to the table and chart recorder
  • Configuring or reconfiguring alarms on channels
  • Saving the configuration
  • Obtaining access to the online help facility.

Usage

To use the program, prepare a configuration file and run the program as follows: 

$DAQROOT/bin/epicsdisplay channelfile

Where:
  • $DAQROOT is the root of the DAQ installation tree. at the NSCL this is /usr/opt/daq/current
  • channelfile is the name of the configuration file you prepared.
