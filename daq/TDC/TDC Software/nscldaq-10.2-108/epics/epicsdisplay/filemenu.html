The File Menu

The file menu contains the following commands:

  • Save... Brings up a dialog that allows you to save the current configuration to file. The saved configuration can be used as the configuration file in subsequent runs of the program.
  • Exit Prompts for confirmation, and if the user confirms, exits the program.
