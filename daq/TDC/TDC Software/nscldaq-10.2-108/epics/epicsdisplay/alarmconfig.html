Configuring Alarms

Double clicking on either the name or comment of a channel allows you to configure alarms on that channel. The alarm configuration dialog box is displayed.

The alarm configuration dialog box is loaded with the current alarm configuration for that channel. If the channel does not have an alarm configuration, the current value of the channel is loaded into the nominal entry. Using the alarm configuration dialog you can

  • Enable or disable alarms for the channel
  • Select a symmetric or an asymmetric acceptable interval about a nominal value for the channel
  • Describe the alarm actions to be taken for alarm conditions on the channel. Note that if you enable the Color in table... action, a color chooser will be displayed to allow you to describe the specific color to be used.
