Configuration Files

The epicsdisplay program requires a configuration file. The configuration file is an ordinary text files. Each line of the file can be either a channel specification or a comment line. Comment lines have a pound symbol (#) as the first character and are ignored by the software. Channel lines begin with the name of the channel followed by commentary information. The commentary information can contain special keywords that are interpreted by the program.

Special comment keywords

Special comment keywords on channel definition lines are interpreted as follows by the program:

  • plotThe channel is added to the strip chart recorder below the table.
  • nominalIntroduces a clause that describes the nominal value window for a channel. When a channel has a value outside the nominal value it is said to be in the alarm state. see the alarmactions keyword below. The nominal window clause has two forms:
    nominal:range[%]

    and 
    nominal+upper[%]-lower[%]

    The first form represents a symmetric interval about the nominal value. The range value is either the half width of the interval or, if an optional percent (%) symbol is appended, the range value represents a half width of the interval expressed as a percentage of the nominal value.
    The second form represents an asymmetric interval about the nominal The upper represents the width of the interval above nominal while lower represents the width of the interval below the nominal. Once again, appending a percent symbol (%) changes the value to a percentage of the nominal value.
  • alarmactions introduces a set of actions to take when the channel is in the alarm state. The allowed actions are any of the following:
    • beep The application periodically beeps when the channel is in the alarm state.
    • popup A popup window is displayed whenthe alarm occurs.

    • color=specThe channel background color is set to spec when it is in the alarmed state. In this case, spec is any legitimate color specification.
