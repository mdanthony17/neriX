Configure -> Set Ramp Speeds...

The Configure -> Set Ramp Speeds... command allows to set the ramp speeds for each channel. Ramp speeds determine how quickly each channel seeks its target when asked to ramp.

The command pops up a modal dialog box. The dialog box has a spin box for the ramp speed for each channel. These spinboxes are loaded with the ramp speeds in effect at the time the dialog is created. Adjust the speeds as desired (units are Volts/Second). Click Ok to accept your modifications. Click Cancel to retain the previous ramp speeds.
