The VHQ Gui

The VHQ gui is divided into several sections.

  • The menu bar provides access to commands that are not directly available onthe control panel.
  • The identification strip, just below the menu bar describes the module controlled by each control panel.
  • Below the identification strip are two Channel Panels each channel panel controls one channel of the VHQ module.
  • Below the channel panels is a log window. The log window displays a log of trip conditions and trip resets. Trips are displayed in red while trip resets are displaye d in green.

Channel panels

The channel panels are how you control each high voltage channel. Each channel panel is clearly labelled with its channel name (A or B). Displays are provided for the current draw in micro-Amps, as well as the voltage the channel is currently putting out.

A spinbox allows you to set voltage targets for ramps. Setting a voltage target does not actually initiate a ramp. Ramps are initiated by clicking the Ramp button below the spinbox providing the target. A fast ramp to 0 can be initiated by clicking the Off button. The ramp speed is a configurable parameter. Each channel also has a configurable configurable current limit.
