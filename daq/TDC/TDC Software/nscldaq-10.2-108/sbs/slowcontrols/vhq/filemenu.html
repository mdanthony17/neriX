The File Menu

The File menu provides commands that read and save configuration and settings files. In addition, commands are provided to close control panels and exit the program.

The File menu provides the following commands:

  • File->New creates a new control panel from a configuration file.
  • File->Read reads a configuration file and applies its setting data to the current control panel
  • File->Write writes the identification connection and settings parameters to a configuration file.
  • File->Close closes the current control panel
  • File->Quit exits the program.
