The Menu Bar

The menu bar is the strip at the top of the control panel. The menus on the menu bar allow you to access commands that are not as frequently used as the device control operations. The menus are:

  • File which allows access to commands that require configuration files, as well as to close control panels and exit the program.
  • Configure which allows you to configure current limits and ramp rates.
  • Help which provides access to the program online documentation
