vhqControl Overview

The vhqControl program allows you to create and run control panels for an arbitrary number of iSeg VHQ 2xx modules.

Usage

The program should be started as follows:

$DAQHOME/Scripts/ControlApplications/vhqControl.tcl [config files...]

Each config file on the command line is a configuration file that describes the module and optionally contains initial settings. If no configuration files are provided, the program will graphically prompt for one. Additional control panels can be created after the program is running as well.

Using the program

The program has extensive help available, and you are just now at the very start of the help pages. Additional help pages describe how to use the program:

  • Using the gui describes how to use the main control panel.
  • The menubar contains many commands that are not directly avaialble on the gui
  • Configuration files are also described in detail.
