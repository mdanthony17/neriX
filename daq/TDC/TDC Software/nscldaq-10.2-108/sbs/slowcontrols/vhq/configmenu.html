Configuration menu

The configuration menu has commands that allow you to configure various parameters of the module under control:

  • Configure->Set Current Limits... Allows you to configure the current limits for each channel of the module.
  • Configure->Set Ramp Speeds... allows you to configure the voltage ramp speeds for each channel of the module.
