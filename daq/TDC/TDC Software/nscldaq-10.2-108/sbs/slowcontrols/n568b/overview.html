Overview

This control panel allows you to setup a CAEN N568b shaping amplifier module connected to this computer via a CAEN high speed VME CAENnet interface installed in any of this computer's VME crates.

  • The Graphical user interface allows you to manipulate the module settings.
  • The File menu allows you access to several useful comands.
  • The Help menu allows access to the panel's online help.

In order to run it is necessary to create a configuration file.
