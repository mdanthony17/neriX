Configuration Files

Configuration files describe the module and, or a group of settings. The module description includes identification meaningful to people, as well as information that describes to the computer how to connect to the module. The settings includes per channel and module global setings.

Configuration files are simpley Tcl scripts. Configuration parameters are just Tcl variables or Tcl array elements.

Module Identification

Module identification parameters are all scalar variables. The following Tcl variables identify the module to humans and the computer.

  • name Provides a human readable name for the module.
  • crate Provides the number of the VME crate in which the CAENnet controller connected to the module is installed.
  • base Provides the VME base address of the controller connected] to the module
  • nodeid
    • Provides the CAENnet node id of the module. This is visible on the front panel of the module.

Settings parameters

The N568B has both module global and per channel settings. The module global settings are stored in Tcl scalar variables. Per channel settings are stored in Tcl array elements indexed with the channel number (an integer from 0 through 15 inclusive):

  • offset contains the common offset for the module. This is a global parameter.
  • finegain(i) is the fine gain for each channel in arbitrary units from 0 through 255 inclusive.
  • coarsegain(i) is the module coarse gain in mV.
  • polezero(i) is the magnitude of the pole zero adjust in arbitrary units from 0 through 255 inclusive
  • outpolarity(i) is the output polarity, zero means positive and one negative
  • outconfig(i) is the output configuration. Zero means direct while one means direct.
  • shapetime(i) is the shaping time in μSeconds. Valid values are 0.1, 1.0, 3.0, and 6.0
