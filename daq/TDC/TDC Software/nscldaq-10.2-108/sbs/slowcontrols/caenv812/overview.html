CAENV812 control panel overview

The CAENV812 control panel allows you to control and arbitrary number of CAEN V812 constant fraction discriminators.

  • The Gui topic describes how to operate the control panel
  • The filemenu topic describes the commands available to you in the application's file menu.
  • The helpmenu topic describes the commands available to you in the application's help menu.
  • The config topic describes how to prepare and interpret the configuration files used by this program.

The control panel can be started as follows: 

$daqroot/Scripts/ControlGui/caenv812control.tcl [configfile...]
Where:
  • $daqroot is where the NSCL Data acquisition system has been installed. at the NSCL, this is /usr/opt/daq/current
  • configfile are configuration files describing the initial set of CFDs you want to control.
