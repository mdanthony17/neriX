The File menu

The file menu contains commands that manipulate configuration files or control the set of control panels displayed by the application.

  • New... allows you to create additional control panels.
  • Read... allows you to read a configuration file into the control panel you are working with
  • Write... allows you to save the configuration and settings of a control panel to file.
  • Close allows you to close the active control panel.
  • Exit allows you to exit the program, closing all control panels
