The CAEN V812 discriminator control panel

The title of the window containig the CAEN V812 discriminator control panel is the name of the module as defined in the configuration file. The CAENV812 control panel window is divided into 6 sections.

  • The top part of the control panel is taken up with the menubar. The menubar has File and a Help menu.
  • Below the menubar is a line of text that identifies the module. This text provides the base address and serial number of the module.
  • Below the identification line is a frame that allows you to select a channel of the discriminator, set its threshold, enable or disable the channel. The Set All copies the selected channels' threshold into all the channels.
  • Below the channel control frame are two frames to control the width and deadtimes of banks of 8 channels. The CAEN V812 has deadtime and width values that are common to groups of 8 channels.
  • Finally, at the bottom of the GUI is a controller for the Majority threshold.

The controls are essentially all 'spinboxes' A spinbox is a text entry with up and down arrows that allow you to modify the value. Holding down an arrow repeatedly increments or decrements the value. You may also type in a new value and either hit the Enter or Tab key for that value to be accepted.
